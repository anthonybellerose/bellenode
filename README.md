# Bellenode

Site web **bellenode.com** — futur application de gestion de stock pour bar/restaurant.

## Stack

- **Backend**: ASP.NET Core (.NET)
- **Base de données**: Microsoft SQL Server
- **Frontend**: à déterminer (Razor / React)
- **Hébergement**: SmarterASP.NET
- **DNS/CDN**: Cloudflare
- **Domaine**: Namecheap

## Structure

- `app_offline.htm` — page de maintenance affichée automatiquement par IIS lorsqu'elle est déposée à la racine du site. Déposer le fichier pour mettre le site hors ligne, retirer le fichier pour le remettre en ligne.

## Déploiement

FTP vers SmarterASP (voir identifiants en mémoire projet privée).
